  - whg_train.csv: 性別・身長・体重データ（全5000サンプル，学習用）
  - whg_test.csv: 性別・身長・体重データ（全5000サンプル，評価用）
  - dmy_train.csv: 人工的に作成したダミーの2次元データ（全1000サンプル，学習用）
  - dmy_test.csv: 人工的に作成したダミーの2次元データ（全1000サンプル，評価用）


・download.py

  画像データをダウンロードするプログラム．次のように実行する．
  python download.py -t=MNIST

  -tオプションにはダウンロードするデータセットの名称を指定する．現状，MNIST, CIFAR10, VGGFace2, Place365 の4種類のみ指定可能．

  MNIST および CIFAR10 は，各々の全データがダウンロードされる．
  VGGFace2 および Place365 は，本来数百万枚ほどのデータを含み，サイズが大きすぎるため，ごく一部のみがダウンロードされる．

  .tar.gz 形式の圧縮ファイルがダウンロードされるため，別途解凍処理が必要．


・MNIST
・CIFAR10
・VGGFace2
・Place365

  それぞれ，対応する .tar.gz をダウンロード・解凍すると，同名のフォルダ生成される．

  - train_data: 学習用画像が格納されているディレクトリ
  - test_data: 評価用画像が格納されているディレクトリ
  - train_list.csv: 学習用画像のファイルパスのリスト
  - test_list.csv: 評価用画像のファイルパスのリスト

  画像のファイル名は，train_data と test_data のぞれぞれで，00000.png, 00001.png, 00002.png, ... のように 5 桁の連番となっている．
  拡張子は Place365 のみ .jpg であり，それ以外のデータセットは全て .png である．
