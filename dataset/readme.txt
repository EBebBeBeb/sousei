[Example]

・内容： サンプルコードで用いる2次元データが保存されているフォルダ

・フォルダ内のファイルの説明
  - whg_train.csv: 性別・身長・体重データ（全5000サンプル，学習用）
  - whg_test.csv: 性別・身長・体重データ（全5000サンプル，評価用）
  - dmy_train.csv: 人工的に作成したダミーの2次元データ（全1000サンプル，学習用）
  - dmy_test.csv: 人工的に作成したダミーの2次元データ（全1000サンプル，評価用）


[download.py]

・内容： 画像データセットをダウンロードするプログラム．

・実行コマンド例
  python download.py -t=MNIST

・実行コマンド例（Google Colaboratoryのセルで実行する場合）
  %run download.py -t=MNIST

・オプション
  -t: ダウンロードするデータセットの名称．
      現状，MNIST, CIFAR10, VGGFace2, Place365 の4種類のみ指定可能．

・備考： MNIST および CIFAR10 は，全データがダウンロードされる．
         VGGFace2 および Place365 は，本来数百万枚ほどのデータを含み，サイズが大きすぎるため，ごく一部のみがダウンロードされる．
         いずれも形式は .tar.gz であり，ダウンロード後，別途解凍処理が必要．


[MNIST]

・内容： MNISTデータセットが保存されているフォルダ．
         MNIST.tar.gz をダウンロード・解凍すると生成される．

・フォルダ内のファイルの説明
  - train_data: 学習用画像が格納されているサブフォルダ
  - test_data: 評価用画像が格納されているサブフォルダ
  - train_list.csv: 学習用画像のファイルパス（MNISTフォルダからの相対パス）のリスト
  - test_list.csv: 評価用画像のファイルパス（MNISTフォルダからの相対パス）のリスト

・備考： 画像のファイル名は，train_data と test_data のぞれぞれで，
         00000.png, 00001.png, 00002.png, ... のように 5 桁の連番となっている．
         拡張子は全て .png である．
         画像のサイズ・チャンネル数の情報は次の通り．
          * 縦幅 28 pixels
          * 横幅 28 pixels,
          * チャンネル数 1（グレースケール画像）
